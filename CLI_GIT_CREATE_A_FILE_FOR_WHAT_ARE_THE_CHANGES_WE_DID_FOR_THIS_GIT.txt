Repository Key Stages: (Local - version of the repo that stored on your computer)

	Staging Area
		CLI: git status --> check the status of the git status
	Modifying Area
		CLI: git add . --> add the changes on the CLI or git local repository.
			       --> Example: You have added changes on this file.txt with a repository v.2
			       --> Once you have made the changes ont the file.txt, try again the "git status" it will show modified.
		or
		CLI: git add "filename" --> add the changes on the CLI or git local repository.
					--> Example: You have added changes on this file.txt with a repository v.2
					--> Once you have made the changes ont the file.txt, try again the "git status" it will show modified.
	Committing Area
		CLI: git commit -m --> Add remarks to the git (local) for the changes on your repository.
